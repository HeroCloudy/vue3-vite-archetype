# Vue 3 Vite 全家桶

整合 Vue3 全家桶 Demo。

### Frameworks

- Vue 3
- Vite 2
- TypeScript
- TSX
- Vue Router
- Pinia
- Element Plus

## Project setup
```
yarn
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```
